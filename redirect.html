Tableau content on dataviz.int.thomsonreuters.com has migrated to the respective Refinitiv and Thomson Reuters infrastructures.

The new URLs are:
Refinitiv Tableau server: https://dataviz.int.refinitiv.com
Thomson Reuters Tableau server: http://dashboards.int.thomsonreuters.com
Both servers have single sign-on and your logon should be seamless, however if you are prompted to provide a user name and password, use your Windows credentials as follows:

On Refinitiv server

User: domain\user id (e.g. ten\u123456)
Password: your windows password

On Thomson Reuters server

User: employee id (without U)/ Contractor id (with C)
Password: your windows password (how you login to your computer)
